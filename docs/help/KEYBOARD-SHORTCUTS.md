# Keyboard Shortcuts

## Overview

Lab Imager currently relies primarily on toolbar and mouse-driven interactions.

Keyboard shortcuts will be introduced as the application matures.

---

## Current Shortcuts

No dedicated application shortcuts are currently implemented.

Standard Windows editing shortcuts remain available where supported.

| Function | Shortcut |
|----------|----------|
| Copy | Ctrl+C |
| Paste | Ctrl+V |
| Cut | Ctrl+X |
| Undo | Ctrl+Z |
| Redo | Ctrl+Y |
| Select All | Ctrl+A |

---

| Stop Recording | Ctrl+Shift+R |

Planned shortcuts are subject to change during development.

---

## Notes

Keyboard shortcuts should not interfere with Notes Editor functionality.

Text-editing shortcuts inside the Notes Editor take priority when the editor has focus.
