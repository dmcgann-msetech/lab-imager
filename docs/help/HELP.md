# Lab Imager Help

## Overview

Lab Imager is a Windows-native microscope and HDMI capture workstation designed for electronics inspection, documentation, evidence collection, and laboratory workflows.

The application combines live camera preview, freeze-frame inspection, evidence capture, technician notes, metadata collection, and recording capabilities into a single workstation-oriented interface.

---

## Common Tasks

### Start Camera Preview

1. Select a camera from the Source dropdown.
2. Select a capture format.
3. Click **Start Preview**.

---

### Freeze an Image

1. Start Preview.
2. Click **Freeze**.
3. Inspect the frozen image.
4. Click **Resume** to return to live view.

---

### Capture Evidence

1. Verify session metadata.
2. Verify technician notes.
3. Click **Capture**.

Generated evidence may include:

* Original PNG
* Evidence PNG
* HTML evidence report
* Future PDF package

---

### Start Recording

1. Start Preview.
2. Click **Record**.
3. Perform inspection activities.
4. Stop recording when complete.

Current recording format:

* AVI
