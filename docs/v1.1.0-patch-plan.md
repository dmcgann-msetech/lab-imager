# Lab Imager v1.1.0 Patch Plan

## Status

Scaffolded. No implementation started.

## Goal

Small post-release patch focused on UI cleanup, runtime safety, update access, and status readability.

## Patch Items

### LI-1101 — Remove Settings Sprocket

Remove the settings sprocket/icon because settings are not planned for v1.1 and would be redundant.

### LI-1102 — Lock Source and Encoding Controls During Unsafe States

Prevent users from changing source, capture format, encoding, or related runtime-sensitive options while preview, recording, freeze, or capture workflows are active.

### LI-1103 — Add Update Entry Point

Add an `UPDATE` or `Check for Updates` entry that opens the GitHub Releases page in the default browser.

No auto-update system in v1.1.

### LI-1104 — Convert Bottom Status Bar to Two-Line Layout

Change the bottom status bar to support two lines so long status output does not overflow off the visible app area.

## Rules

- No broad refactor.
- No settings module.
- No installer work.
- One patch item per commit.
- Build after every patch item.
- Update roadmap/changelog before merge.
